# TenMINAtor
TenMINAtor es la librería que extiende opciones de cuantizacion para correr modelos en local; -Con capacidades de re y entrenamiento para grandes LLMs, formada con independencia para proceso Autónomo.

Version: [Prueba](http://tenminator-votpvbfn.manus.space): (No funcional) 


Web oficial: [http://wAI.cAm](http://WAI.cam)


La libreria TenMINAtor es el modelo especifico relacional, usado para el entrenamiento y re-entrenamiento de grandes o pequeños modelos de Procesamiento del Lenguaje Natural, (LMs); esencial para toda la traduccion en lenguajes terricolas muy conocidos y sus medios de comunicacion; como el Canal Nou, el Bisca Television o beoutQ. 

Pese a ser un proyecto libre requiere la verificacion minima; como la visual del Espejito Magico; o similar de identificacion de usuarios. 

Para usarse en local con toda la formacion del modelo de traducción de lenguajes, requiere la descarga de la minima APP con conexion, a la version del Modelo General de Lenguajes y de Multi Traducciones  entre Especies; para poder usarse en un entrenamiento de su funcion.


Si desean realizar otros entrenamientos especificos distintos, les sugerimos personalizar librerias especificas como [TenMiniTorch](http://github.com/yoqer/TenMiniTorch) ; como la base para cualquier otro proyecto; o la General extensa que entrena Gradientes y todas las funciones generalizadamente; a la que pueden añadir cualquier Framework especifico de entrenamiento que usen: [TenMinaTorch](http://github.com/yoqer/TenMinaTorch) : la libreria general no personalizada para funcion del proposito comunicativo y sus validaciones razonales: `TenMINator`

la que tiene activas ciertas limitaciones instrumentales y puede usarse con la APP simple que pueden ver en este enlace en Version [Prueba](http://tenminator-votpvbfn.manus.space).
